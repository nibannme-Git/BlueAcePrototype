## BlueAce プロトタイプ集


BlueAceのプロトタイプを二つアップロードしました。prototype1.imgはカーネルのテストで、prototype2.imgはGUIのテストです。

## 起動


起動にはQEMUまたはv86が推奨されます。

# prototype1.img


QEMUの場合、qemu-system-i386 -fda prototype1.imgで起動します。

# prototype2.img


QEMUの場合、qemu-system-i386 -fda prototype2.imgで起動します。
何回か再起動しないと正しく表示されない場合があります。
